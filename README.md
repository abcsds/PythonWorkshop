# Tasks
 1. Open a binary file on a text editor
 2. Open a text file with a different file extension than `.txt`
 3. `text/LittlePrince.txt`:
  - Find all occurrences of the word "tame"
  - How many times does the word "day" appear?
  - Change all occurrences of the word "king" to "fool"
  - Find a spelling error (US English)
  - Correct all occurrences of a spelling error
 4. Make `text/DragonsInLiterature.txt` a python list
 5. Add a key binding for Atom's `editor:split-selections-into-lines` command
 6. Open a terminal:
  - `cd` into directory
  - Start the Python interpreter
  - Multiply two numbers
  - Concatenate two strings
  - Assign a variable
 7. Strings:
  - Concatenate two strings
  - Replace spaces with underscores
  - Slice string(s)
  - Format a string
 8. Lists:
  - Create a list
  - Append two lists
 9. `for` loops:
  - Print all numbers from 1 to 100
  - Print all squares of the numbers from 1 to 10
  - Print the last task, ten times.
  - Print all the squares of the numbers from 0 to 10, but only if the number is even
 10. Functions:
  - Define a function for the power of 1.5
  - Define a function that replaces spaces in a file name for underscores
